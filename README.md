# Docker for React Application

The purpose of application is to develop the production ready react application with docker and travis ci application.

<img width="1431" alt="Screen Shot 2022-10-25 at 1 23 57 AM" src="https://user-images.githubusercontent.com/33331778/197698366-0025c79f-1211-4d12-ae2f-057e446114a0.png">

The Application contains the 2 container, one containing the code to host the application and other container is to test the test cases. All the test cases 
in the test node will be tested by travis, if sucessful the application will be loaded into the AWS Elastic BeanStalk.
